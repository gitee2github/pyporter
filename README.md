# pyporter

#### Description
A rpm packager bot for python modules from pypi.org

#### Installation

1.  python3 setup.py install

#### Preparation
Install below software before using this tool
gcc
gdb
libstdc++-devel
python3-cffi

#### Instructions

1.  pyporter <package> -s -b -d -o python-<package>.spec
